# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-11-29

### Added
-   **Event Tracking**: Create and track custom events.
-   **Counters**: Increment and decrement event counts.
-   **Persistence**: Data is saved locally using AsyncStorage.
-   **Delete Confirmation**: Custom modal to prevent accidental deletions.
-   **Haptics**: Haptic feedback on interactions (Mobile only).
-   **Animations**: Smooth scale animations on button press.
-   **Dark Mode**: System-wide dark mode with toggle and persistence.
-   **Cross-Platform**: Support for Android, iOS, and Web.
